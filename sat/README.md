# SAT

Esta sección contiene documentos oficiales y guías técnicas relacionadas con el Régimen de Factura Electrónica en Línea (FEL) emitido por la Superintendencia de Administración Tributaria (SAT) de Guatemala.

Existen dos documentos esenciales que debemos de tener conocimiento acerca de:

El primer documento titulado como `guia-formato-grafico.pdf`, detalla y contiene los elementos obligatorios que deben incluirse en la representación gráfica de cualquier Documento Tributario Electrónico (DTE), como facturas, notas de crédito/débito, recibos, entre otros.

El segundo documento titulado como `reglas-validaciones-FEL.pdf`, contiene información varia sobre los diferentes tipos de DTE's que existen y ademas aborda temas como las validaciones y reglas de negocio que deben cumplirse para que un Documento Tributario Electrónico (DTE) sea certificado por la SAT.

Es de suma importancia tener conocimiento acerca de estos dos documentos para poder trabajar con los sistemas de la empresa.
